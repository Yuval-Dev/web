Recursion's most practical use in the field of computer science is to break down a task that is way to big to be done (in many cases, this could be due to the task having a time complexity that scales up too fast or the task simply being too complex to be done iteratively or explicitly) into many smaller tasks that are easier to be done. An example of this is with some mathematical functions (from which recursion originated. more on that) in which: in order to find the next element, you must do some arithmetic onto the previous element(s).
