Beyond The BootLoader

After making your bootloader, you will need to compile it with NASM like this: nasm.exe -f bin -o .bin .asm and then run it with qemu like this: qemu-system-x86_64.exe -fda .bin which should then greet you with the message you put in your code!
After making this simple bootloader, the next step would be to enter 32 bit mode and run the later stages of your kernel/OS/program. Osdev is a very good wiki website for learning more about this as well as reference for making such things.
